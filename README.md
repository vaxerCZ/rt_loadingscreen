# RT LOADINGSCREEN

A modern and clean loading screen featuring a welcome screen, video background, music player, and multiple customizable sections.

---

Click the link below to see the loading screen in action:

👉 **[Watch Showcase](https://www.youtube.com/watch?v=-DVYX45wsyo)**  

---

## 🎨 Features
- Welcome screen with **interactive button**
- **Video background** support  
- Built-in **music player**  
- Custom **links section**  
- Optional **gallery view**  
- Editable **server name & description**  
- Multiple **pre-made color themes**  
- Clean and modern UI  

---

## ⚙️ Customization
You can easily customize the loading screen in the `config.js` file:

- Change **welcome message and button**
- Replace or edit **background video**
- Customize **music and volume**
- Edit **links, gallery, and sections**
- Change **server name and description**
- Switch between **color themes**

Make it fit perfectly to your server style.

## 📦 Installation
1. Download or clone this repository  
2. Add the resource to your `resources` folder  
3. Configure everything in `config.js`  
4. Start/restart your server
